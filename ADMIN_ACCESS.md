# 管理画面アクセスガイド

## 🔐 管理者専用アクセス

YNU-TWIN の管理画面は**管理者専用**です。一般ユーザーには表示されません。

## 📍 アクセス方法

### 開発環境

```
http://localhost:8080/ynu-admin-panel-secure-9x7k2m
```

### 本番環境

```
https://your-domain.com/ynu-admin-panel-secure-9x7k2m
```

**🔒 セキュリティ重要**: このURLは秘密にしてください！
独自のURLに変更する方法は `SECURITY_ADMIN.md` を参照してください。

## 🚀 起動手順

### 1. バックエンドを起動

```bash
cd backend
.\venv\Scripts\Activate.ps1
python -m api.main
```

または

```bash
cd backend
.\venv\Scripts\Activate.ps1
uvicorn api.main:app --reload
```

### 2. フロントエンドを起動

```bash
# プロジェクトのルートディレクトリで
pnpm dev
```

### 3. 管理画面にアクセス

ブラウザで以下のURLを開く：
```
http://localhost:8080/ynu-admin-panel-secure-9x7k2m
```

**🔒 このURLは秘密です！** 他の人に共有しないでください。

## 🔒 セキュリティに関する注意

### 現在の状態

- URLを知っていれば誰でもアクセス可能
- 認証機能は実装されていません

### 推奨されるセキュリティ対策（今後の実装）

1. **Basic認証の追加**
   - ユーザー名とパスワードでの保護

2. **IPアドレス制限**
   - 特定のIPアドレスからのみアクセス可能

3. **認証トークン**
   - JWTトークンによる認証

4. **ファイアウォール設定**
   - 本番環境でのアクセス制限

## 📋 よくある質問

### Q: ユーザー画面から管理画面へのリンクはありますか？

**A:** いいえ、ありません。管理画面はURLを直接入力してアクセスする必要があります。

### Q: ブックマークしておけますか？

**A:** はい、管理者はブラウザにブックマークしておくことをお勧めします。

```
名前: YNU-TWIN 管理画面
URL: http://localhost:5173/admin
```

### Q: パスワードは必要ですか？

**A:** 現在のバージョンでは認証機能はありません。本番環境で使用する場合は、適切なアクセス制限を設定してください。

### Q: 管理画面のURLを変更できますか？

**A:** はい、可能です。**独自のURLに変更することを強く推奨します！**

`client/App.tsx` の `ADMIN_SECRET_PATH` を変更：

```typescript
// 現在
const ADMIN_SECRET_PATH = "/ynu-admin-panel-secure-9x7k2m";

// 変更例（独自のランダムな文字列に）
const ADMIN_SECRET_PATH = "/your-own-secret-path-x9k2m5p7";
```

詳細な手順は `SECURITY_ADMIN.md` を参照してください。

### Q: 複数の管理者でアクセスできますか？

**A:** はい、URLを知っている人なら誰でもアクセスできます。ただし、同時編集の競合管理は実装されていないため、注意が必要です。

## 🛡️ セキュリティのベストプラクティス

### 開発環境
- localhost なので外部からアクセスできない
- そのまま使用して問題ありません

### 本番環境
- ⚠️ **必ず認証機能を追加してください**
- 推奨事項：
  1. Basic認証の実装
  2. HTTPSの使用
  3. ファイアウォールの設定
  4. アクセスログの記録
  5. 管理者IPアドレスの制限
